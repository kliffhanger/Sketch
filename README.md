# React + Vite
A canva type clone with Fabric JS and Material UI 
(First time usage)

libraries used- Fabric Js, Material UI

React Context Api used

Key Features
- Add Line
- Add Circle
- Add Rectangle
- Add Polygon
- Add Text
- Upload Image
- Draw on canvas
- Delete selected Item or select group of items
- Clear canvas
